Sujet

Résumé

Le but du TP est le développement de différents shaders :
  • Matériau spéculaire : shader de Phong
  • Matériau métallique : shader métal
Le code fourni correspond à un matériaux parfaitement diffus.

Question 1 : Calcul de l'éclairement par pixel

Le code fourni effecte le calcul de l'éclairement/la couleur dans le vertex shader. Le fragment shader ne faisont pas de calcul supplémentaire.Transformez le code afin que le calcul de l'éclairement s'effectue dans le fragment shader à partir de la position et de la normale calculées dans le vertex shader. Faites en sorte de pouvoir passer d'une version à l'autre via une checkbox:
< input id = "per-fragment" checked= "checked" type= "checkbox" > Eclairage par pixel (fragment) (/ par sommet (vertex))

Question 2 : Shader Phong

A partir du code fourni, vous ajoutez la composante spéculaire du matériau (Cf. Figure 1).


Figure 1: Matériau spéculaire.

Pour cela, vous ajouterez des uniform aux shaders : un coefficient de diffusion uKd et un indice de spécularité uSpec.

// material : color, Kd, spec
uniform vec3 uMatColor ;
uniform float uKd ;
uniform float uSpec ;

Le terme spéculaire est donné par :
Is = Ks . Isource . cosn(r,v)
avec :
Ks = 1-Kd (Kd: coefficient de diffusion),
n : indice de spécularité,
r : vecteur donnant la direction de réflexion de la lumière,
v: vecteur donnant la direction de la vue.

Rappel
r = reflect(p,n) donne la direction de reflexion de p selon le vecteur n.
